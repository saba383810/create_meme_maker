# プリセットテンプレート

このフォルダに画像を置いて `manifest.json` に登録すると、ツールの左パネルに「プリセット」として表示されます。

## 追加手順

1. 画像ファイル（png / jpg / webp / gif）をこのフォルダに置く
2. `manifest.json` の `templates` 配列にエントリを追加する

```json
{
  "templates": [
    { "file": "sample_template.png", "name": "サンプル（2コマ）" },
    { "file": "my_image.png", "name": "表示したい名前" }
  ]
}
```

- `name` は省略可（省略時はファイル名を表示）
3. commit して push すると、GitHub Pages のビルド後に反映される

## 注意

- **public リポジトリに置いた画像は全世界に公開・再配布されます。** 自分に権利がある画像、または再配布が許諾されている画像だけを置いてください。
- プリセットは初回クリック時に「マイテンプレート」へコピーされます。文字枠のレイアウトはコピー側に保存されるので、このフォルダの画像は差し替えない限りそのままで大丈夫です。
- プリセット機能は http(s) 配信時のみ有効です（`file://` で直接開いた場合は表示されません。`python3 -m http.server` を使ってください）。
