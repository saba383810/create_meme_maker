# クソミーム・メーカー

漫画のコマなどのテンプレート画像に文字を重ねて、クソミーム画像を作るローカルGUIツール。
サーバー不要、`index.html` をブラウザで開くだけで動きます。スマホ・PC共通のUIです。

## 使い方

```bash
open index.html
```

うまく動かない場合（クリップボードコピーが失敗する等）はローカルサーバー経由で:

```bash
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

## 3ステップの流れ

1. **えらぶ** — テンプレート画像を選ぶ（新規追加 / マイテンプレート / プリセット）
2. **つくる** — キャンバス上でテキスト枠を配置し、画面下のエディタで文字を入力
3. **できた** — できあがりをプレビューして 共有 / PNG保存 / コピー

ヘッダーのステップバーでいつでも行き来できます。

## 機能

| 機能 | 説明 |
|---|---|
| テンプレート追加 | 「画像からつくる」/ 画像のドラッグ&ドロップ / `Ctrl+V` 貼り付け（どの画面でも可） |
| プリセット | `templates/manifest.json` に登録した画像が「プリセット」欄に表示され、タップで取り込み（詳細は [templates/README.md](templates/README.md)） |
| テキスト枠 | 画像上をドラッグして作成。白塗り背景で元の文字を隠して新しい文言を重ねる |
| 縦書き対応 | 漫画のフキダシ用。「ー」「…」等の回転、「、。」の位置調整も対応 |
| リアルタイムプレビュー | 文字を打つとその場でキャンバスに反映 |
| 共有 | スマホでは共有シートからLINE・X等へ直接送れる（Web Share API 対応ブラウザ） |
| PNG保存 | 「できた」画面の `PNG保存` or `Cmd/Ctrl+S`。スマホは画像長押しでも保存可 |
| クリップボードコピー | `コピー` ボタン（Chrome / Edge / Safari 対応） |
| 自動保存 | テンプレート（画像＋枠レイアウト＋文言）はブラウザの IndexedDB に自動保存 |

## 操作方法（つくる画面）

- **枠の作成**: 画像の空白部分をドラッグ、または「＋ 枠を追加」
- **移動**: 枠をドラッグ
- **サイズ変更**: 選択中の枠の四隅ハンドルをドラッグ
- **拡大表示**: 二本指ピンチ（スマホ）/ トラックパッドのピンチ・`Ctrl`+ホイール（PC）。二本指ドラッグで移動、「全体表示」ボタンで戻る
- **枠の切替**: 画面下のチップをタップ、またはキャンバスの枠をタップ
- **文字編集**: 画面下のテキスト欄。枠のダブルクリックでもフォーカス
- **スタイル**: ツールバー（サイズ / フォント / 色 / 配置 / 削除）から設定
- **削除**: ツールバーの「削除」/ 枠を選択して `Delete`
- **微調整**: 矢印キーで1px移動（`Shift` 併用で10px）

各枠ごとに フォント（ゴシック/明朝/丸ゴシック）・サイズ・太字・文字色・背景色・縁取り・配置・縦書き を設定できます。

## 注意

- データはブラウザ内（IndexedDB）に保存されます。`file://` で開いた場合と `http://localhost` で開いた場合では保存領域が別になります。
- テンプレートに使う画像の著作権・利用範囲は各自で確認してください。
