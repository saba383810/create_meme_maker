# クソミーム・メーカー

漫画のコマなどのテンプレート画像に文字を重ねて、クソミーム画像を作るローカルGUIツール。
サーバー不要、`index.html` をブラウザで開くだけで動きます。

## 使い方

```bash
open index.html
```

うまく動かない場合（クリップボードコピーが失敗する等）はローカルサーバー経由で:

```bash
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

## 機能

| 機能 | 説明 |
|---|---|
| テンプレート追加 | 「追加」ボタン / 画像のドラッグ&ドロップ / `Ctrl+V` 貼り付け |
| テキスト枠 | 画像上をドラッグして作成。白塗り背景で元の文字を隠して新しい文言を重ねる |
| 縦書き対応 | 漫画のフキダシ用。「ー」「…」等の回転、「、。」の位置調整も対応 |
| リアルタイムプレビュー | 文字を打つとその場でキャンバスに反映 |
| PNG保存 | `PNG保存` ボタン or `Cmd/Ctrl+S` |
| クリップボードコピー | `コピー` ボタン（Chrome / Edge / Safari 対応） |
| 自動保存 | テンプレート（画像＋枠レイアウト＋文言）はブラウザの IndexedDB に自動保存 |

## 操作方法

- **枠の作成**: 画像の空白部分をドラッグ、または「＋枠を追加」
- **移動**: 枠をドラッグ
- **サイズ変更**: 選択中の枠の四隅ハンドルをドラッグ
- **削除**: 枠を選択して `Delete` / 右パネルの「枠を削除」
- **微調整**: 矢印キーで1px移動（`Shift` 併用で10px）
- **文字編集**: 右パネルのテキストエリア。枠のダブルクリックでもフォーカス

各枠ごとに フォント（ゴシック/明朝/丸ゴシック）・サイズ・太字・文字色・背景色・縁取り・配置・縦書き を設定できます。

## 注意

- データはブラウザ内（IndexedDB）に保存されます。`file://` で開いた場合と `http://localhost` で開いた場合では保存領域が別になります。
- テンプレートに使う画像の著作権・利用範囲は各自で確認してください。
